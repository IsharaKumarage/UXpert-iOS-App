# UXpert – AI-powered UI/UX Design Feedback App

**UXpert** is an iOS app designed to help UI/UX designers, students, and freelancers improve their interface designs. The app leverages **Core ML** with the **FastViT model** to analyze uploaded UI screenshots and provide actionable feedback on:

- Color harmony
- Layout balance
- Contrast and accessibility
- Typography and spacing

It also stores feedback history locally using **CoreData**, and visualizes trends with **SwiftUI Charts**.

---

## Features

- Upload or select UI screenshots for analysis
- Real-time AI-based design critique
- Dashboard for visualizing improvement over time
- History of past uploads and feedback
- Dark mode and light mode support
- Animated, modern, and responsive UI using SwiftUI
- Accessibility support (Dynamic Type, high contrast)

---

## Technologies Used

- **SwiftUI** – Frontend UI framework
- **Core ML** – Machine learning integration (FastViT model)
- **CoreData** – Local persistence for history and results
- **SwiftUI Charts** – Visualize feedback trends
- **MVVM Architecture** – Clean separation of concerns
- Optional: **FLAN-T5 Base** for summarizing textual feedback (if integrating text summaries)
